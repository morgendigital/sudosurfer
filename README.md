## Getting Started

1. Clone our repository:
    ```bash
    git clone https://github.com/morgendigital/sudosurfer.git
    ```

2. Make the script executable:
    ```bash
    chmod +x surf.sh
    ```

3. Execute the script:
    ```bash
    ./surf.sh
    ```
